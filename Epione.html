Epione - Application Web

Epione
  1. Projets
  2. Epione

Creation d'une application Web pour la societé DeepOr

Dans le cadre académique on a réalisé une application Web à l'aide de 3 frameworks integré. Epione est un portail de prise de rendez-vous médical avec un ajustement intelligent des rdv en cas de désistement du patient. L’outil se compose d’une application mobile pour les patients et d’une application web pour les médecins. Java EE pour l'exposition des Services Web REST . Asp.NET pour cosommer ces services là et exposer encore des services pour les consommer en Angular 5
