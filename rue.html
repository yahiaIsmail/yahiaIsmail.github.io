Smart Street Projet Embarqué Académique

Dans le cadre académique on a réalisé une application Desktop pour controller une maquette intelligente d'un rue intelligent que l'on a réalisé aussi . en utilisant QtCreator (C++) , on a réalisé une interface Administrative pour la municipalité pour controller des capteurs situés au sein d'un rue intelligent que l'on a simulé avec une maquette intelligente qui se compose de plusieurs capteurs pour automatiser plusieurs operations dans un rue tels que le fonctionnement de l'arrosage des arbres dans ce rue ,etc.. a l'aide d'un carte Arduino Uno

Toutes les operations sont enregistrées dans une base de données MySQL

  • Client : Municipalité Ariana Soghra
  • Type : Application Desktop & Maquette embarquée
